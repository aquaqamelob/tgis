# TGiS Quiz

Quiz z pytań kolokwialnych — źródło: `pytania.txt` (PDF profa).

Odpowiedzi są **losowo przemieszane** przy każdym pytaniu. Zaznaczasz 2 poprawne zdania (prawdziwe lub fałszywe, zależnie od pytania).

## Lokalnie (Bun)

```bash
cd quiz
bun install
bun dev
```

## Odświeżenie pytań z txt

```bash
cd quiz
python3 scripts/extract-from-txt.py
```

## Vercel

Root Directory: `quiz` → Deploy.
